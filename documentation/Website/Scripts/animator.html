animator.js

makeFrom(animator, i)

This function is responsible for making svg that connect the lines. These are the ones that the lines will start from.
It takes animator and i as arguments, animator is used for adding ids for the svgs and i is for the indexing of animator.

concatLog(to, From, messageString)

This function is responsible for adding new text from the log, the arguments are formated and put into the log. Every time it is called it autoscrolls to the bottom of the log.

makeTo(animator, i)

This function is responsible for making svg that connect the lines. These are the ones that the lines will end at.
It takes animator and i as arguments, animator is used for adding ids for the svgs and i is for the indexing of animator.

makeDeploymentRelations(animator)

This function is responsible for making the relations between the different devices in the deployment diagram. The relations are responsible for making the lines that connect the different devices.
The argument for this function is the animator and it is used for makeFrom and makeTo that is called in this function.

createDeploymentClass(animator, i, left, top)

This function makes the the boxes for the deployment diagram. It uses animator for knowing what to add to the class/box, i is an index for knowing what info to take from the animator and top and left is used for deciding where the boxes will be on the screen.

createDeploymentLines(animator, i)

This function is responsible for making lines between the different devices in the deployment diagram. It takes the the id of two different svgs and draw a line between the two different svgs.
The argument is used for finding the correct devices too connect with lines.

createNodes(object, frameToAppend)

This function is responsible for parsing the JSON File nodes in its entirety. It takes two parameters, a JSON Object (this could also be an array) as well as the frame to append the nodes to.
It is called recursively after each node creation and stops once the parsing is done. The initial JSON must have at least one "send" node otherwise the function will hang

arrayifyString(string)

it takes a string and puts it into an array. It splits the string every times a comma is found and out the rest of the string into a new index of the array as long as no more commas are found.
